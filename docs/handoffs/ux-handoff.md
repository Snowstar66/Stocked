# UX Handoff

Datum: 2026-05-19

## UX-Riktning

UI följer framingens "Enterprise control plane" trots konsumentdomän: tät informationsyta, tydliga statusar, tabell/listskanning, högerrail med prioritering och händelser, samt små radier och neutrala blå ytor.

## Nyckelflöden

- Starta i dashboarden, inte på landningssida.
- Lägg till vara via kompakt formulär.
- Se varan i kategoriserad lageröversikt.
- Använd "Snart-dags" för prioriterad handling.
- Få måltidsidéer som stödjer användning av befintliga varor.
- Skapa inköpslista och undvik dubblettinköp.
- Radera lokal data med bekräftelse.

## Textprinciper

Svensk text använder åäö. Datumtexter undviker säkerhetsråd och talar om planering: "Använd snart", "Planera in" och "Inte ett livsmedelssäkerhetsråd."

## Kvarvarande UX-Risk

Ingen faktisk användartestning eller skärmläsartestning är genomförd. Layouten är CSS-responsiv, men bör kontrolleras i minst mobil och desktop före release.
