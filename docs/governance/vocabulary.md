# Vocabulary

| Term | Definition | Källa | Notering |
|---|---|---|---|
| MatSvinnskollen | MVP-app för hushållens lokala matöversikt. | Framing package | Produktnamn. |
| Vara | En matvara som användaren registrerar manuellt. | EPIC-001 | Kallas inte produktpost från extern databas. |
| Kategori | Lokal gruppering: Kyl, Frys, Skafferi, Annat. | EPIC-001 | Egen kategori stöds inte i denna MVP. |
| Bäst före / använd före | Användarens planeringsdatum för prioritering. | EPIC-002 | Inte livsmedelssäkerhetsbedömning. |
| Snart-dags | Lista över varor med datum inom sju dagar. | SC-003 | Visar prioritet, inte säkerhet. |
| Måltidsidé | Regelbaserad enkel idé baserad på befintliga varor. | EPIC-003 | Inte receptdatabas, allergiråd eller näringsberäkning. |
| Inköpslista | Manuell lista över saknade varor. | EPIC-004 | Ingen butikskoppling. |
| Lokal data | Appstate sparat i browser localStorage. | EPIC-005 | Ingen molnsynk eller konto. |
